Getting Token:

Head over to Discord's Developer Portal and create a new application.
There's A TON of Tutorials therefore I'm not going into how to create an app.
Once you've gotten a valid token from your new bot application and
the bot in your server, paste Token in line 14 of ez_captcha.py.
-------------------------------------------------------------------------------------------
Setting Up Server:

This system relys on @everone Role having NO PERMS beside viewing the "Verification Channel"
Nothing Else...!!!!

Role:
You'll need to create a role that every member needs in order to use server/channels.
This Role will be automatically assigned to new members who complete the captcha.
This role needs to be made now, then copy the ID into Line 11 of ez_captcha.py.
See "Getting ID's"* (below)

Channels:
Next, we need to make 2 new channels:
- One, [Verification Channel], that only @everyone can see.
This will be where new users are verified

- Two, [Welcome Channel], that all memebers can see.
This channel will recieve notifications when a new user
has been verified and is typically done in the "Main Chat".
-------------------------------------------------------------------------------------------
Gettings ID's:

Next, head over to your Discord Server as we'll need to get some required info.
Go to User Settings > Appearance then Enable Developer Mode.

To pull an object ID or "Snowflake ID",
we'll right-click on, for instance, a user/channel name, which , if dev-mode enabled
will show Copy ID at the dropdown. Take these ID's and drop them in
ez_captcha.py in the correct locations as directed in the source.
